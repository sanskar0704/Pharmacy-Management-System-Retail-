About This Project

This Pharmacy Management System is a comprehensive web application designed to streamline pharmacy operations including inventory management, billing, and sales tracking. The system provides an intuitive interface for managing medicines, processing sales, and generating reports.

Built with modern web technologies, this system offers real-time inventory tracking, automated billing, and detailed sales analytics to help pharmacy owners manage their business efficiently.

Sanskar Surag

Developer & Creator

Full-stack developer passionate about creating efficient and user-friendly applications.

Features

  • Medicine inventory management with stock tracking
  • Real-time billing and sales processing
  • Sales history and reporting
  • Low stock alerts
  • Customer information management
  • Export functionality for sales data
  • User authentication and role-based access

Technology Stack

Backend Python Flask
Database SQLite
Frontend HTML5, CSS3, JavaScript
API RESTful API
